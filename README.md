# Beautypixer 🦋

✨ É uma extensao de navegador inspirada pelo [Momentum](https://momentumdash.com/) e [Pexels](https://chromewebstore.google.com/detail/pexels/fdnpgodccdfgofepkclnopmbnacjkbnj)
com o diferencial de ser completamente gratuita e mesclar algumas coisas de ambos.

🛠️ Essa extensão foi desenvolvida usando:
- [Svelte](https://svelte.dev/): Framework para desenvolver a interface
- [Vite + TS](https://vitejs.dev/): Bundler
- [DM Sans](https://fonts.google.com/specimen/DM+Sans): Familia de fontes

💖 e muito amor e carinho envolvido.
